# Secure File Encryption System
### BIT4138 — Advanced Cryptography Practical Project

## Overview
A Python-based secure file encryption system implementing:
- Classical Ciphers (Caesar, Vigenere)
- Stream Ciphers (LFSR, RC4)
- AES-256-CBC File Encryption
- RSA-2048 Public Key Cryptography
- Secure Password Hashing (PBKDF2)

## Tech Stack
- Python 3.x
- Cryptography Library
- OpenSSL

## Project Structure
secure-file-encryption/
├── keys/           # Key storage (not uploaded to GitHub)
├── caesar.py       # Caesar cipher
├── vigenere.py     # Vigenere cipher
├── lfsr.py         # LFSR generator
├── rc4.py          # RC4 stream cipher
├── aes_encrypt.py  # AES file encryption
├── aes_decrypt.py  # AES file decryption
├── rsa_keygen.py   # RSA key generation
├── rsa_encrypt.py  # RSA encryption
├── rsa_decrypt.py  # RSA decryption
└── login.py        # Password hashing system
